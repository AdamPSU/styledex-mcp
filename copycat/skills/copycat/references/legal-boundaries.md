# Legal and ethical boundaries

Copycat captures publicly visible design patterns for reference. It does not produce official brand systems and should not imply affiliation with the source site.

Allowed:

- observing typography and spacing
- observing color relationships
- observing border, radius, shadow, and layout patterns
- describing visual mood and density
- creating original UI inspired by those patterns

Not allowed:

- copying logos or brand marks
- copying exact source copy
- copying proprietary images, icons, illustrations, or videos
- recreating exact page layouts section by section
- presenting generated @DESIGN.md files as official brand documentation
- bypassing authentication, paywalls, or bot protections

When a user asks for a direct clone, redirect to a style-focused outcome. Ask what they want to build and use the captured style as inspiration for original work.

When applying a saved style, keep the user's product identity and content distinct from the source website.
